  • None of the above

2. How to prevent equipment fire?

  • Make sure that the equipment is off at all times
  • Never check if the equipment is defected
  • Do not use the equipment so it doesn’t overhea
  • Have the equipment regularly checked by an expert or competent person

3. How to prevent electrical fire?

  • Use as much devices you can per outlet
  • Always install and repair electrical equipment by yourself
  • Put extension cords under the carpet
  • Get a qualified electrical contractor to install and/or repair electrical equipment

4. How to prevent arson fire?

  • Encourage unauthorised entry into the building
  • Never use signs
  • Keep flammable items properly stored and secured
  • Do not use any form of alarms e.g. intruder alarms

5. How to prevent flammable material fire?

  • Make sure there are a lot of flammable items in the building
  • Reduce the amount of flammable materials to the smallest amount possible
  • Replace low flammable materials with high flammable ones
  • All of the above

6. How to limit the development of fire?

  • Leave all of the doors and windows open
  • Never use fire extinguisher
  • Keep all doors and windows closed
  • Try reducing the development of fire using your hands

7. How to put out storage fire?

  • Smoother with paper to cut off the air
  • Use type A or/and B extinguisher according to the type of fire
  • Use type S extinguisher
  • Use type C extinguisher

8. How to put out electrical fire?

  • Keep the plug in the outlet
  • Try to cool it with water
  • Use type C extinguisher
  • Use type B extinguisher

9. How to put out cooking fire?

  • Keep the stove on until the fire is gone
  • Use type C extinguisher
  • Use type B extinguisher
  • Use type A extinguisher

10. Which step does apply when using fire extinguisher?

  • Pull the trigger
